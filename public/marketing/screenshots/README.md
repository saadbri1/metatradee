# Marketing screenshots

Real MetaTradee product screenshots go here, generated by
`scripts/capture-marketing-screenshots.mjs` from a **seeded demo account** (clearly
non-real sample data) at 2x DPR in the dark theme.

**Nothing in this folder is committed as fabricated UI.** Until real screenshots
are captured, the marketing sections render token-built, `aria-hidden`,
number-free device motifs — never fake dashboards with invented figures.

To generate:

```bash
MARKETING_SHOTS_BASE_URL=https://staging.metatradee.app \
DEMO_EMAIL=demo@metatradee.app DEMO_PASSWORD=… \
node scripts/capture-marketing-screenshots.mjs
```

Then swap the decorative `PreviewMotif` / `DashboardPreview` for `next/image`
(AVIF/WebP, responsive `sizes`, `priority` only on the hero image, blur
placeholder, intrinsic dimensions to keep CLS at 0).
